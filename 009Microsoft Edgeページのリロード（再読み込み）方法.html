Microsoft Edgeページのリロード（再読み込み）方法

Microsoft Edgeページのリロード（再読み込み）方法は、以下の手順でリロード（再読み込み）できます。

  • キーボードで操作する場合
  • ブラウザで操作する場合
  • 複数のタブを更新する場合

キーボードで操作する場合

  1. Microsoft EdgeでWebページを表示します
  2. ファンクションキーの［F5］キーを押すと再読み込みます。

ブラウザで操作する場合

  1. Microsoft EdgeでWebページを表示します。
  2. 画面左上の［ リロード（最新の情報に更新）］をクリックします。

複数のタブを更新する場合

  1. Microsoft EdgeでWebページを表示します。
  2. 2.複数のタブを開いた状態で、いずれかのタブを右クリックし、表示された一覧から［すべてのタブを最新の情報に更新］をクリックし
    ます。

以上で「お気に入りを表示させる方法」は完了です。
