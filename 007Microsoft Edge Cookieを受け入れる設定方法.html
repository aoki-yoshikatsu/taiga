Microsoft Edge Cookieを受け入れる設定方法

Microsoft Edge Cookieを受け入れる設定方法は、以下の手順で設定できます。

Cookieとは

ユーザの利用環境などを記憶・保存することにより、ホームページ上でのサービス等を便利に利用できるようにする
仕組みです。
  1. 画面右上の［設定など］をクリックします。
  2. ［設定］をクリックします。
  3. ［詳細設定の表示］をクリックします。
  4. Cookieの［Cookieをブロックしない］を選んでクリックします。
